# Smart Constructor

## API (Express)

### Run (dev)

```bash
npm install
npm run dev
```

### Run (prod)

```bash
npm install
npm start
```

### Endpoint

- `GET /ping` → `{ "message": "pong" }`

### Env

- `PORT` (default `8080`)
- `HOST` (default `0.0.0.0`)
